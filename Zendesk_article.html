The Zendesk Connector allows you to download source content for the following content types and upload translations to and from Zendesk.

  • Guide module articles, sections, and categories.

  • Support module dynamic text.

Use Case

There are several use cases for connectors.

  • The project manager can add files directly to the project from the online repository.

  • Set up an automation widget that allows senders to add files to requests directly from the online repository.

  • Use Automatic Project Creation (APC) to have Memsource automatically create a new project when a change in the last edit date (edited_at attribute) is detected in a monitored file or an entire list. If the connector is configured to create jobs only for languages that are flagged for translation, APC will only create jobs for translations of flagged articles.

    • The selected file will be imported the first time you run APC.

  • The Connector API allows you to automate the steps you perform manually through the UI. You can use webhooks to let Memsource notify third-party systems about certain events (such as job status changes).
